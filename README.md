# Multiscale Transcriptomic Network Models of Parkinson's Disease at the Single-Cell Level

## Overview

This repository implements the **MetaCell-based Multiscale Network Analysis (MCMNA)** framework, a comprehensive workflow for constructing and analyzing multiscale transcriptomic networks from single-nucleus RNA sequencing (snRNA-seq) data.

<img width="2550" height="1822" alt="2025_04_23_PD Network workflow_v4" src="https://github.com/user-attachments/assets/ecf4192b-e66e-4936-8b32-570a49b23e99" />

The framework includes:

* Metacell construction and characterization
* Biologically meaningful correlation (BMC) analysis
* Gene co-expression network analysis
* Bayesian causal network analysis
* Key driver analysis

MCMNA was applied to a large snRNA-seq dataset generated from postmortem substantia nigra (SN) samples from **23 idiopathic Parkinson's disease (PD) cases** and **9 healthy controls** (average age: 81 years), as previously described by Wang et al. (*Science Advances*, 2024; PMID: 38198537). The study profiled **315,867 high-quality nuclei** and identified **12 distinct cell populations**.

## Workflow Summary

**Workflow Name:** MetaCell-based Multiscale Network Analysis (MCMNA)

**Language:** R

**Type:** Multiscale transcriptomic network analysis workflow

### Description

Single-cell transcriptomic datasets are inherently sparse, making robust network inference challenging. MCMNA addresses this limitation by aggregating transcriptionally similar cells into **metacells**, thereby reducing sparsity and improving signal-to-noise ratio.

The workflow enables:

* Construction of robust gene-gene interaction networks
* Identification of cell-type-specific co-expression modules
* Integration of differential expression and causal network analyses
* Systematic prioritization of disease-associated key driver genes

---

## Workflow Steps

### 1. snRNA-seq Data Preprocessing

Preprocessing follows the published pipeline from: https://github.com/mw201608/PD_substantia_nigra

### 2. Metacell Construction

Generate metacells from individual nuclei within each cell type and disease condition.

### 3. Biologically Meaningful Correlation (BMC) Matrix Generation

Identify significant gene-gene correlations while accounting for cell abundance and sparsity.

### 4. Metacell-Based MEGENA Analysis

Construct multiscale gene co-expression networks using the MEGENA framework.

### 5. Network Pruning and Filtering

Apply pruning strategies to remove spurious network connections and improve network specificity.

### 6. Metacell-Based Bayesian Network Construction

Infer causal regulatory relationships among genes.

### 7. Key Driver Analysis

Identify candidate causal regulators and top disease-associated driver genes.

## Repository Contents

### 1. Metacell Construction

#### 1.1 `Covariate_correction.R`

Corrects covariate effects in the original single-cell dataset prior to metacell construction, such as Age, Sex, Postmortem interval (PMI)

#### 1.2 `Metacells_seurat.R`

Generates metacells using the **SuperCell** R package within each cell type and condition.

---

### 2. Biologically Meaningful Correlation (BMC) Matrix Generation

#### 2.1 `Cluster_no_perm_dir.R`

Calculates correlations for all possible gene pairs.

#### 2.2 `Truncated_distribution_test_signed.R`

* Fits truncated normal distributions
* Calculates mean squared error (MSE)
* Plots MSE versus minimum cell cutoff
* Determines optimal cutoff thresholds
* Estimates background correlation distributions

#### 2.3 `Cluster_sig_dir.R`

Identifies significant gene-gene correlations at the optimal minimum cell cutoff threshold (FDR < 0.05).

#### 2.4 `BMC_generation.R`

Generates the final BMC matrix by adjusting significant correlations using the estimated background distribution.

---

### 3. Metacell-Based MEGENA Analysis

#### 3.1 `Cluster_sorted.R`

Submits MEGENA jobs using absolute correlation values sorted by magnitude.
